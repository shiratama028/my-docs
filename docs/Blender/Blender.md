# 基本操作関係
| 項目              | 内容                                                                                                                                   |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| シーンのレンダー設定などを保存 | レンダーのサンプル数などを128程度にした状態でファイル＞デフォルト＞スタートアップファイルの保存<br>次回起動時も同じ設定で開かれるようになる                                                            |
| - 画面の4分割        | ビュー＞エリア＞画面の4分割<br>もう一度4分割を押すともとに戻る<br>※これだとmayaのように各ビューで設定を変えることができない<br>それぞれのビューで設定を変えたい場合は手動でエリアを分割してあげないといけない                     |
| 単位の設定           | 初期はメートル<br>シーン設定からUnit Scaleを調整<br>└ベースはメートル１=1m　0.01=1cm                                                                            |
| 自動保存            | Preferences＞Save＆Load＞Auto save<br>デフォルトでon<br>もし強制終了などで落ちた場合、File＞RecoveryからAuto Saveで復元できる可能性がある<br>WindwsのTempフォルダなので削除しなければここにある |
| ファイル保存時の圧縮設定    | Preferences＞Save＆Load＞Compress File                                                                                                  |
